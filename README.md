# Go-Auth

This is just a simple demo project I'm working just for fun, to learn more about Go and Gin.

For the "watch" while developing, I'm using `air`.

```
go install github.com/air-verse/air@latest
```

Create a `.env` file on project root with these vars:

```
JWT_SECRET=your_jwt_secret
POSTGRES_USER=your_db_user
POSTGRES_PASSWORD=your_password
POSTGRES_DB=your_db
```
